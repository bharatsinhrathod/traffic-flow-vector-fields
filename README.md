# Traffic Flow Analysis using Vector Fields
SEM-4 Vector Calculus Practical Project

## Description
This project models vehicle movement on a road network using 2D vector fields.
Divergence, Curl, and Line Integrals are used to detect congestion and optimize paths.

## Libraries Used
- Python 3.x
- NumPy
- Matplotlib
- SciPy

## How to Run
pip install numpy matplotlib scipy
python traffic_flow.py
